# SP変換リファレンス — More Friends 全12ページ

**作成目的**: PCワイヤーHTMLからSPワイヤー設計に必要な情報を高密度に抽出したナレッジファイル。  
**参照ルール**: `03_SPワイヤー専用ルール.md` を設計判断の正本とする。本ファイルは構造サマリーのみ。

---

## 凡例

| 記号 | 意味 |
|---|---|
| ✅ 表示 | SP でそのまま表示 |
| 🔽 縮小 | 情報量・サイズを縮小して表示 |
| ❌ 非表示 | SP では非表示 or 省略 |
| 🔁 ループ | CMS ループ対象（カスタム投稿 or クエリ） |
| 🔒 固定 | 編集者が変更しない想定の領域 |
| ✏️ 編集可 | 管理画面から編集できる想定の領域 |
| ⚠️ 注意 | SP変換時に判断が必要なポイント |
