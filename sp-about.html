More Friendsとは、
どんなサービス？

ペットシッターに安心を持てない方へ。 More Friendsが大切にしていることをお伝えします。

WHAT IS

ペットシッターって、何が違うの？

ペットホテル・一般的なペットシッター・More Friends。 それぞれの特徴をカードで比較できます。

More Friends

ペットの個性に寄り添う、信頼関係ベースのシッター

  • ✓ 自宅でシッティング。環境の変化なし
  • ✓ 無料カウンセリングで関係づくりから始める
  • ✓ ペット1頭ずつの個性・ペースを尊重
  • ✓ シッティング中の写真・動画報告
  • ✓ 飼い主さんとの継続的なコミュニケーション
一般的なペットシッター
  • ✓ 自宅でシッティング。環境の変化なし
  • △ 担当者の対応・経験はまちまち
  • △ 初回面談・関係づくりのプロセスは業者次第
  • △ 報告の頻度・質にばらつきがある場合も
ペットホテル
  • ✗ 見知らぬ環境・他のペットと同空間
  • ✗ ストレスを感じやすいペットには向かない場合も
  • ✓ 施設が充実していれば安心感は高い
  • △ スタッフの目は届いているが個別対応は限定的
PROMISES

More Friendsの3つの約束

  • 01

    慣れるプロセスを飛ばさない

    初回は必ず無料カウンセリングから。ペットさんが 「この人は安全だ」と感じるまで、時間をかけます。 いきなりシッティングを押しつけることはしません。

  • 02

    その子だけのペースを尊重する

    犬種・猫種・年齢が同じでも、性格はまったく違います。 マニュアルではなく、目の前のペットさんに合わせた ケアをお届けすることを約束します。

  • 03

    飼い主さんを不安にさせない

    「今どうしてるかな」と心配させないために、 シッティング中の様子を写真・動画・テキストで こまめにご報告します。

PHILOSOPHY

「友達になる」という考え方

ペットシッターは「預かり屋」ではないと思っています。

ペットさんにとって、シッターは「家族以外の安心できる人」。 そういう存在になれたとき、ペットさんも飼い主さんも、 どちらも穏やかな時間を過ごせると信じています。

More Friendsという名前には、 「ペットさんに、もっと友達を増やしてあげたい」 という思いが込められています。

フィロソフィービジュアル（16:9）
TRUST

安心は、段階を踏んでつくるもの

  1. 1

    まず顔を見せる

    無料カウンセリングでご自宅を訪問。 ペットさんに「知らない人」から「知ってる人」へ。

  2. 2

    短い時間から始める

    お試しシッティングは短時間から。 ペットさんが「いなくても大丈夫」と感じるペースで進めます。

  3. 3

    関係を育てる

    回数を重ねるごとに信頼が積み上がります。 ペットさんが玄関で待つようになったら、それが答えです。
